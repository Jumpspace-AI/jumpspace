# Support

Jumpspace is currently experimental but usable.

## Questions

For usage questions, start with:

- `README.md`
- `docs/src/content/docs/getting-started/index.md`
- `docs/src/content/docs/agents/using-with-agents.md`
- `docs/src/content/docs/reference/cli.md`

If the docs do not answer the question, open a GitHub issue with the `question` label.

## Bugs

When reporting a bug, include:

- Jumpspace version: `jumpspace --version`
- Install freshness: `jumpspace release install-doctor --json`
- Command that failed.
- JSON output or error envelope when available.
- Whether the repo was initialized with `jumpspace init`, `init --auto`, or `bootstrap apply`.

## Security

Do not report security issues in public issues. Follow `SECURITY.md`.
